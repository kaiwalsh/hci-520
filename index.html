Introduction

Welcome to this e-learning system! This lesson will teach you how to secure and optimize your home network. Between improved performance, faster speeds, and (hopefully) less networking issues, there are some benefits all tech users can gain with a few simple steps.

Learning Objectives

After this lesson, you should able to:

  • Identify common security vulnerabilities in home networks.
  • Execute procedures that fix security and performance problems on a home network.
  • Implement techniques to troubleshoot unknown issues.
  • Compare performance details and specifications of networking hardware.
  • Critique the effectiveness of other security practices.
